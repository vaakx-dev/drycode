# Starter chat exploration gallery

Twenty-five throwaway views produced by five independent design explorations for `docs: shape the starter chat experience`.

## Run every exploration

```sh
node prototypes/starter-chat-explorations/server.mjs
```

Open <http://127.0.0.1:4180/> and select any view.

Each exploration also has its own README and server. All views use the same conceptual capabilities and in-memory state while testing different information structures. They intentionally exclude editors, file trees, terminals, source control, and other IDE features.
