# Settings variation 07

A standalone Drycode Settings exploration focused on the local control plane: the composed Extension Graph, frozen Service Slots, Runtime health, Drycode Home, provider extensions, Shell-owned preferences, and Reload. It keeps the starter chat's full-width topbar and session-oriented sidebar while making Settings the primary surface.

## Run

Open `index.html` directly, or serve the repository with any static server. Lucide icons are loaded from the pinned browser CDN.

## Interactions

- Jump between settings sections using the sticky Control plane index.
- Select Extension Graph nodes to inspect manifest and slot details.
- Run diagnostics, copy a report stub, change Shell preferences, inspect providers, clear temporary data, and stage a provider extension.
- Collapse the sidebar and open the Reload confirmation/progress flow.
