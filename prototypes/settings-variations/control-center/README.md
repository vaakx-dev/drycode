# Drycode Control Center

A standalone settings-page variation for the Drycode architecture-led control center. Overview is the default surface, with internal navigation for the resolved Extension Graph, provider configuration, Runtime Generation, Data & Logs, and About.

## Run

Open `index.html` directly, or serve the repository with any static server. The prototype uses the pinned Lucide browser CDN and LobeHub's white provider marks; interactions are browser-only stubs.

## Included interactions

- Internal section navigation and collapsible Drycode sidebar
- Provider configuration dialogs
- Runtime Reload confirmation and progress state
- Diagnostics, log, manifest, and path action feedback
