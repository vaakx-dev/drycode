# Drycode Settings variations

A review gallery containing ten independent Settings-page implementations integrated with the selected starter chat shell.

- `control-center`: architecture-led control center
- `ownership-first`: architecture-led ownership model
- `variation-03` through `variation-10`: independent design explorations

## Run the gallery

```sh
node prototypes/settings-variations/server.mjs
```

Open <http://127.0.0.1:4190/>. Select a direction from the gallery sidebar, use the arrow controls to move between variations, or open any direction in a separate tab.
